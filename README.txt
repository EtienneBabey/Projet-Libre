
Pour lancer le jeu il faut extraire le fichier Build Unity.zip dans un dossier
et exécuter le fichier Snake.exe.
-----------------------------------------------------------------------------
Pour accéder aux scripts il faut aller dans le fichier scripts.zip, il y en a trois.
-----------------------------------------------------------------------------

Pour contrôler les directions prises par le serpent dans le jeu il est possible 
d'utiliser soit les touches ZQSD ou soit les touches directionnelles du clavier.
Ce jeu reprend le même principe que le jeu Snake, donc lorsque le serpent touche
un mur de face et où son propre corps la partie est finie et renvoie le joueur
au menu du jeu. De plus lorsque le serpent mange une pomme il grandit est la 
difficulté augmente, le but étant de manger le plus de pomme possible.

---------------------------------------------------------------------
Ce jeu ne fonctionne que sous Windows 
et il est possible qu'il y ait quelques 
problèmes graphiques selon les résolutions
de l'écran utilisé.
